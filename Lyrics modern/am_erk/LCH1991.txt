Wie machen es denn die Schneider?
So machen sie’s: Hier ein Läppchen, da ein Läppchen,
geben zuletzt ein Kinderkäppchen.
So machen sie’s, so machen sie’s!

Wie machen es denn die Bäcker?
So machen sie’s:
Sie beten das Vaterunser: das beste Brot ist unser.
So machen sie’s.

Wie machen es denn die Metzger?
So machen sie’s:
Sie halten große Hunde und wiegen kleine Pfunde.
So machen sie’s.

Wie machen es denn die Wirte?
So machen sie’s:
Sie nehmen die Kreide wohl in die Hand
und schreiben doppelt an die Wand.

Wie machen es denn die Schmiede?
So machen sie’s:
Sie schmeißen das Eisen rund und dumm,
und endlich kommt der Bauer drum.

Wie machen es denn die Schreiner?
So machen sie’s:
Sie nehmen den Hobel wohl in die Hand
und drücken die Mädchen an die Wand.
So machen sie’s.
