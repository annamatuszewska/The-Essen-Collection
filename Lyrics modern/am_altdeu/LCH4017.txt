„Ich kenne ein schönes Mädchen, ich möchte sie gerne zur Frau haben; Wenn sie von Haselnussbäumen die kleinen Fäden spinnt.“

„Soll ich dir von Haselnussbäumen die kleinen Fäden spinnen, so musst du mir von Lindenlaub ein Paar Kleider schneiden.“

„Soll ich dir von Lindenlaub ein Paar Kleider schneiden, so musst du mir die Schere holen aus der Mitte des Rheins.“

„Soll ich dir die Schere holen aus der Mitte des Rheins, so musst du mir eine Brücke schlagen aus einem kleinen Reis.“

„Soll ich dir eine Brücke schlagen aus einem kleinen Reis, so musst du mir das Siebenstern zu hoher Mittagszeit zeigen.“

„Soll ich dir das Siebenstern zu hoher Mittagszeit zeigen, so musst du mir die Gläserburg auf einem Pferd reiten.“

„Soll ich dir die Gläserburg auf einem Pferd reiten, so musst du mir die Spuren schlagen auf glattem Eis.“

„Soll ich dir die Spuren schlagen auf glattem Eis, so musst du sie aber deine Füße tragen im heißen Sonnenschein.“

„Soll ich sie deine Füße tragen im heißen Sonnenschein, so musst du mir einen Schweißbecher bringen von Wasser und von Wein.“

„Soll ich dir einen Schweißbecher bringen von Wasser und von Wein, so musst du mir den Grabstein zu Pfeffer mahlen.“

„Soll ich dir den Grabstein zu Pfeffer mahlen, so musst du mir alle wilden Schweine in eine Höhle treiben.“

„Soll ich dir alle wilden Schweine in eine Höhle treiben, so musst du mir deine Mutter geben für ein Mädchen zur Frau.“

„Soll ich dir meine Mutter geben für ein Mädchen zur Frau, so sollst du sieben Jahre hängen und wieder lebendig werden; die Teufel aus der Hölle können dich nicht vertreiben.“
